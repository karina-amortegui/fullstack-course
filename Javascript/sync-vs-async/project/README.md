# PROJECT: SIMULATED API CALL WITH A LOADING INDICATOR

Goals:
    - Show a "Loading..." message while fetching data.
    - Display the fetched data after a delay.

## Included in the project:
    * Uses async / await to simulate an API request.
    * Displays loading while fetching data.
    * Uses set timeout to simulate this delay.
    * Handles air with try / catch

## Takeaways
    - Javascript is single-threaded, but we can run tasks asynchronously.
    - setTimeout and setInterval allow delayed exevution.
    - Promises with .then and .catch handle asynchronous operations.
    - Async / await provide a cleaner way to work with asynchronous code.
