## CONDITIONALS: IF STATEMENTS

Control the flow of a program by executing different blocks of code, based on conditions.

* If statemnt: executes a block of code if the condition is true.

## IF ... ELSE IF ... ELSE STATEMENTS

* Condition checks each statement and stops as soon as one is true.
* Ternary Operator ( ? : ) - shorthand "if else" statement

## FOR LOOPS

* Loops repeat a block of code multiple times.
* For Loop is most common and used when the number of iterations is known.

## WHILE LOOP

* Use when the number of iterations is unknown.

## DO WHILE LOOPS

* do ... while (Executes at least onced).
* The loop executes once before checking the condition.
