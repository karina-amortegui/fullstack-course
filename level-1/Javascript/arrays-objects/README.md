## ARRAYS
- variable that stores multiple values in a single variable.
- Arrays are zero indexed, meaning zero is the first item.
- Finding the length of an array: 
    * can use .length to get the number of elements.
- "Push" adds a new element to the end of an array.
- "Pop" removes the last element from an array.
- "Shift" removes the first element (index 0) from an array.
- "Unshift methods adds an element to the beginning of an array (index 0).

## ARRAY METHODS 
Splicing
    - Splice can do everything push, pop, shift, and unshift can do.
    - Removes and/or inserts elements.
Methods
    - Creates a new array with elements that match a condition.
Map
- transforms the elements inside the array.
- Creates a new array by applying a function to each element or callback to each element.
