# PROJECT: A STUDENT MANAGEMENT SYSTEM

Goals:
- allow the user to add students
- allow the user to remove students
- allow the user to filter students who scored above a certain grade
- allow the user to map student data into a formatted list

# What we did
- used splice to remove a student by name
- used filter to find who scored above a given grade
- used map to formats student data into a readable list

# Takeaways
- Arrays store multiple values and support various methods (push, splice, map, etc.)
- Objects store key value pairs and allow easy data management
- Looping through arrays and objects is easy wiht forEach map and forIn
