## WHAT IS A FUNCTION?
    - A block of reusable code that performs a task when called.
    - Help reduce repitition and make code more manageable.

## ARROW FUNCITONS AND FUNCTION EXPRESSIONS
    - function expressions (aka anonymous functions) store a function inside a variable.
    - a modern way to write functions.

### Curly Braces
    - needed for multiple lines.
    - return must be explicit inside the curly braces.


## FUNCTION SCOPE AND HOISTING
    - Scope: local vs global variables. Global - available everywhere, local - exisit inside a function only.

### Hoisting
    - function declaration vs expressions.
    - Declarations (hoisted) can be called before definition.
    - Expressions (like anonymous and arrow functions are not hoisted).
